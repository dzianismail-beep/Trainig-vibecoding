# Private Investor Landing Page

A professional landing page for private investors featuring biography and contact form.

## Features
- Responsive design that works on all devices
- Professional gradient background and modern styling
- Biography section with professional photo and statistics
- Interactive contact form with validation
- Smooth scrolling navigation
- Social media links

## Files
- `index.html` - Main landing page
- `tetris.html` - Bonus Tetris game (legacy)

## Deployment
This site is ready to be deployed on GitHub Pages or any web hosting service. Simply upload the HTML file and it will work immediately.

## Customization
To customize the page for your needs:
1. Replace the profile image URL with your own photo
2. Update the biography text with your information
3. Modify the statistics to reflect your investment record
4. Update contact information and social media links
5. Customize colors by modifying the CSS gradient values

## Contact Form
The contact form includes client-side validation and is ready to be connected to a backend service or email handler.
