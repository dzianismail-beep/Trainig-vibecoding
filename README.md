# Tetris Game

A simple Tetris game built with Python and Pygame.

## Requirements
- Python 3.x
- Pygame

## Installation
1. Install Python 3 from https://www.python.org/
2. Install Pygame:
   ```powershell
   pip install pygame
   ```

## Running the Game
Run the following command in your terminal:
```powershell
python tetris.py
```

## Controls
- Left Arrow: Move left
- Right Arrow: Move right
- Up Arrow: Rotate
- Down Arrow: Move down faster
- Space: Drop piece

Enjoy playing Tetris!
